Privacy Policy – TrackU Collect

Last updated: 24.11.2025

TrackU Collect is an experimental beta application distributed via TestFlight for private testing purposes only. This privacy statement explains how the app processes data in accordance with the EU General Data Protection Regulation (GDPR) and applicable German data protection laws (BDSG).

1. Controller (Verantwortliche Stelle)

Philipp Baumann
Loofile Innovation UG (haftungsbeschränkt)
Email: dev@comebackplus.de

2. Data the app collects

The app collects the following information solely for testing and training analysis:

  • Sensor data: accelerometer and gyroscope values
  • Training-related information: number of repetitions, optional notes on exercise quality
  • Optional images: a photo of the training machine (users should avoid including people)

No data is linked to an identifiable person.

3. How data is processed

  • All data is processed only locally on the device.
  • No automatic transmission or background data transfer occurs.
  • Data is shared externally only if the user chooses to manually export it.

4. No personal data

The app does not collect or process: names, email addresses, identifiers, IP addresses, advertising IDs, location data, health data, or any information that can identify an individual.

5. Legal basis (Rechtsgrundlage)

Data processing is based on voluntary consent (Art. 6(1)(a) GDPR). By participating in the TestFlight beta and using the app, testers consent to this processing.

6. Data sharing

No data is shared with third parties. No analytics, tracking tools, or external SDKs are used.

7. Data retention

All data remains stored on the user's device and is removed when the user deletes it or uninstalls the app. If testers export data outside the app, they retain full control over it.

8. Rights of the data subject (Betroffenenrechte)

Testers may:

  • request access to data stored on their device;
  • request deletion of exported data;
  • withdraw consent at any time;
  • file a complaint with a German data protection authority.

Contact: dev@comebackplus.de
