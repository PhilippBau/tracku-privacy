Privacy Policy – TrackU Collect

Last updated: 24.11.2025

TrackU Collect is an experimental beta application distributed via TestFlight for private testing purposes only. This privacy statement explains how the app processes data in accordance with the EU General Data Protection Regulation (GDPR) and applicable German data protection laws (BDSG).

1. Controller (Verantwortliche Stelle)

Philipp Baumann
Loofile Innovation UG (haftungsbeschränkt)
Email: dev@comebackplus.de

2. Data the app collects

The app collects the following information solely for training analysis and testing the beta functionality:

  • Movement and sensor data: accelerometer and gyroscope values
  • Timestamps: time points of sensor readings and repetitions
  • Anonymous user ID: an internally generated identifier that does not contain personal data
  • Repetition data: number of repetitions performed
  • Optional notes: user-provided comments about the quality or characteristics of the recording
  • Quality score: a calculated rating of movement quality from 0 to 100
  • Optional images: a photo of the training machine (users should avoid including people)

No information collected by the app is linked to an identifiable person.

3. How data is processed

  • All collected data is processed exclusively on the user’s device.
  • No automatic data transmission, cloud syncing, or background uploads occur.
  • Data is only shared externally if the user explicitly exports it (e.g., via email or file export).

4. No personal data

The app does not collect or process: names, addresses, phone numbers, email addresses, device identifiers, IP addresses, advertising IDs, precise location data, health data, biometric data, or any information that could identify an individual.

5. Legal basis (Rechtsgrundlage)

Data processing occurs with voluntary consent (Art. 6(1)(a) GDPR). By participating in the TestFlight beta and using the app, testers consent to this processing.

6. Data sharing

No data is shared with third parties. The app does not use analytics services, tracking technologies, or external SDKs that transfer data.

7. Data retention

All data remains stored locally on the user’s device. It is deleted when the user removes it manually or uninstalls the app. If data is exported manually, users maintain full control over how long it is stored.

8. Rights of the data subject (Betroffenenrechte)

TestFlight testers may:

  • request access to data stored on their device;
  • request deletion of any exported data;
  • withdraw consent at any time by stopping use of the app;
  • file a complaint with a German data protection authority.

Contact for rights requests: dev@comebackplus.de

9. Changes to this policy

Any updates to this privacy policy will be published on this page.
